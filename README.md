## 🧠 AI Feature (Core Highlight)

This project includes a custom AI scoring algorithm:

### 📊 AI Score Formula


AI Score = (Priority × Weight) + Urgency Score + Time Factor


### ⚙️ How it works:
- Tasks with higher priority are ranked higher
- Tasks closer to deadline get boosted score
- Shorter tasks get slight advantage
- System automatically sorts tasks
